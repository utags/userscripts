## üöÄ Installation

### Prerequisites

1. Install a userscript manager (choose one):
   - [Tampermonkey](https://www.tampermonkey.net/) (recommended)
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [ScriptCat](https://scriptcat.org/)

### Install Scripts

1. Click an install link in the table (Greasy Fork / ScriptCat / GitHub Raw)
2. Confirm and enable it in your userscript manager
3. Refresh the target site to take effect
